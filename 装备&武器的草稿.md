# 机制
## 锻造
定义：将一个物品B（主要为材料）附加在物品A上  
格式：A+B, (costLevel)=A(..,B) 或者 A+B=*None* -> (A, B) （锻造失败）  
效果1：当 B 为 *技能符文* 且 A 为 **武器** 类，则用 B 对应的技能代替 A 的技能（如 A 的 **Q键技能** 为 *技能1*，B对应技能为 **Q键技能** *技能2*，则A的 **Q键技能**（技能1） 被替换为B对应的技能（技能2））  
效果2：当 B 与 A 为同类型 **工具** 时，则锻造产物 C 为 A 的属性、$min(A_{耐久}+B_{耐久}, A_{耐久max})$ 的耐久与A、B 的 **物品类型**决定  
花费：需要 $2^{稀有度-1}+锻造次数$ **等级**  
***若上述条件都不满足，则锻造失败，不消耗任何材料或等级***

## 附魔
定义：给 **工具** 类添加 *魔咒*  
格式：A+b (costLevel)=A(..,b) 或 A+b=*None* -> (A, b)（无附魔能力、魔咒不适应或超过魔咒数量限制）  
限制：一个工具最多只能有3个 *魔咒*，超过则无法附魔该 *魔咒*  
效果1：当 A 已经拥有与 b 同类的 *魔咒*，合并后的 **魔咒等级** $^{1}$ 未超过 **该魔咒最高等级限制**，则成功附魔  
效果2：当 b 为 *附魔之星* 且 A 为 **工具** 类，则 A 的附魔词条列表增加 b  
花费：需要 $\lfloor 1.5^{魔咒等级}+附魔词条数量\rfloor$ 等级
***若上述条件都不满足，则锻造失败，不消耗任何材料或等级***

# 装备
### 三日月护身符
*在那充斥着时间碎屑的地方，你将看着它们的贪婪被月影所反噬*

触发条件：使用者 *受伤*
触发概率：97%
效果：对攻击发动者造成使用者受到伤害的 $1\%$ ~ $100\%$ （记为 $x$） 的伤害，使用者将只受到本次伤害的 $100\%-x$的伤害  
当反伤比例恰好为 $15.0\%$ 时，则额外反弹 $500\%$ 的伤害  
若连续3次（判定为30秒内） $x<10%$，则下一次必定 $x>50$  
升级：可升级 **额外反弹**，一级：$13.00\%\leq x \leq 17.00\%$；二级：$10.00\%\leq x \leq 20.00\%$（额外倍率仍为500%）  
可升级 **触发概率**，一级：$98\%$；二级：$99\%$

##### 反伤
反伤无视护甲、格挡等，不参与任何额外的伤害计算（也就是真实伤害）

### 四叶草
*一株稀有的异变种，传说它有带来幸运的魔力*  

效果：装备时增加1%的掉落概率
升级：可升级 **掉落概率**，一级：1.5%，二级：3%，三级：5%

##### 掉落概率
如 A 的掉落概率为 98%，则意味着 A **有 98% 的概率掉落1个**  
如 B 的掉落概率为 101%，则意味着 B **有 100% 的概率掉落1个**，另 **有1%的概率再掉落1个**

### 焦虑之眼
*一个不安的眼睛，通过释放微弱的电流让佩戴者感到急迫*  

记忆时长：5秒  
攻速转化率：5%  
速度转化率：1%  
效果：装备时，根据从记忆时长前到此刻累计受伤量和区域难度计算 *不安值*；设累计受伤量为 $x$，区域难度为 $k$，公式为：$a=x\times \frac{k^2}{25}$；设 *不安值* 为 $a$，将攻速提升至原有的 $1.05\times a$，将速度提升至原有的 $1.01\times a$

### 恐惧之眼
*一颗令人恐惧的眼球，散发着血腥的味道*

血量需求：$\leq 3\%$  
反制概率：98%  
击退大小：$2.43 \times 10^5$ N
效果：当血量不足3%时，每间隔 $\frac{1}{6}$ 秒以98%的概率尝试对周围任意一个生物造成1点伤害的 ***重击***，并带有巨量 **击退**
